CivicPulseHub

CivicPulseHub is a web-based complaint management system that allows citizens to report civic issues such as road damage, garbage problems, water leakage, and streetlight failures. The platform helps government officers track and resolve complaints efficiently.

Citizens can submit complaints and track their status, while officers and administrators manage and update complaint progress through dashboards.

Features

Citizen complaint submission

Complaint tracking system

Officer complaint management

Admin dashboard

Secure authentication

Department-based officer assignment

Future improvements include AI-based complaint categorization, priority detection, and automated officer assignment.

Tech Stack

Frontend: React, Vite, TypeScript
Backend: Node.js, Express.js
Database: MongoDB
Security: JSON Web Token, bcrypt
Tools: Git, GitHub, Visual Studio Code

Run the Project

Backend:

cd backend
npm install
npm run dev

Frontend:

cd frontend
npm install
npm run dev
